# Fairy-Wren-Diet-Analysis
Senior thesis project focused on bioinformatics and fecal metabarcoding of fairy wren fecal samples.

Directory

Fairy-Wren-Diet-Analysis.Rproj - R project file for this work

Classifiers - Contains the classifiers for the R pipeline

misc - Contains miscellaneous files (at present, contains only the BRC submission table)

Pipeline_exploration - contains files related to my first attempt to understand and work through the QIIME2 pipeline

reads - contains the reads from the BRC facility sequencing (too large to push at the moment)

zippedreads - contains the same reads as 'reads' but is zipped for data transfer (too large to push at the moment)

pipeline_semifinal - contains my attempt at writing a final pipeline for the QIIME 2 analysis
